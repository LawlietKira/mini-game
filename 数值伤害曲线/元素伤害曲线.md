# 元素伤害曲线（html）

## 需求

1. 用html生成一个曲线图
2. 横坐标为时间，单位为秒
3. 纵坐标为伤害，可以为浮点值，但是显示在坐标轴上的的必须是整数
4. 生成4个曲线，火、土、水、金，每个曲线可单独配置曲线的点位信息
5. 曲线以直线相连
6. 可以对改动后的曲线信息进行保存，保存为json结构，保存在浏览器缓存中，保存时，用console.log打印保存数据
7. 默认数据用json保存，以便后续替换

## 补充规格

1. **默认曲线数据** — 按结构随意生成，后续替换
2. **单条曲线点位数量** — 可任意配置（动态增减点位）
3. **点位交互方式** — 图表上拖拽（拖拽时重新渲染曲线）
4. **图表标题** — `元素伤害曲线估算`

## 技术选型（暂定）

| 项目 | 选型 |
|------|------|
| 图表库 | ECharts（支持拖拽、中文友好） |
| localStorage 键名 | `element-damage-curves` |
| 曲线颜色 | 火=红、土=棕、水=蓝、金=金 |
| 异常处理 | 缓存损坏或无缓存时回退默认数据 |
