## 偷窃小游戏 — 开发规范文档

## 一、游戏概述

玩家在NPC身后进行偷窃操作，通过按键控制偷窃目标与偷窃行为。需在NPC警觉度满值前或未被视野发现的前提下，将偷窃进度累积至目标值以完成偷窃。

---

## 二、核心机制

### 2.1 道具系统

- NPC身上携带若干道具，进入偷窃状态后，玩家 **只能看到道具数量和品质颜色**，无法看到具体道具名称。
- 道具共有 **6种品质**：白色、绿色、蓝色、紫色、橙色、未知。
- 每种品质对应不同的偷窃进度需求值和警戒倍率（详见参数表）。

### 2.2 偷窃进度

- 玩家 **长按 F** 进行偷窃，偷窃进度随时间持续增长。
- 进度增长速率 = `偷窃速度 × 偷窃速度加成系数`（单位/秒）。
- 当偷窃进度 ≥ 当前目标道具品质对应的 `进度需求值` 时，**偷窃成功**。
- 玩家 **松开 F** 后，偷窃进度以 `偷窃回退速度`（单位/秒）匀速降低，最低降至 0。
- 切换偷窃目标时，偷窃进度 **立即清零**。

### 2.3 目标切换

- 玩家长按 **Q**（上一个）或 **E**（下一个）循环切换NPC身上的道具目标。
- 切换目标时，偷窃进度立即清零；当前已累积的警觉度保持不变。
- 长按1秒才能切换目标，可配置

### 2.4 警觉度系统

- 玩家 **长按 Q / E / F 任一按键** 时，NPC警觉度开始匀速增长。
  - 警觉度增长速率 = `警戒速度 × 当前目标品质的警戒倍率`（单位/秒）。
- 玩家 **松开所有操作按键（Q/E/F全部松开）** 后，警觉度以 `警戒回退速度`（单位/秒）匀速降低，最低降至 0。
- 当警觉度 ≥ `警戒值上限` 时，**偷窃失败**。

### 2.5 NPC视野检测

- NPC拥有一个 **锥形视野**，会周期性左右摆动观察。
- 视野检测判定规则：
  - 玩家处于NPC锥形视野范围内 **且** 正在执行偷窃操作（长按F）→ **偷窃失败**（被发现）。
  - 玩家处于NPC锥形视野范围内 **但已松开所有按键** → **不被发现**，游戏继续。
  - 玩家处于NPC锥形视野范围内，长按Q/E切换目标但未按F → **不被发现**（仅触发警觉度增长，不触发视野发现判定）。

### 2.6 偷窃速度

- 实际偷窃速度 = `偷窃速度 × 偷窃速度加成系数`（单位/秒）。
- 偷窃速度加成系数始终生效，无需按键触发，且不影响敌人警觉度增长。

### 2.7 环境参数

- 新增 `环境参数`，作为额外系数直接乘到敌人基础警觉度增长值上。
- 实际警觉度增长 = `警戒速度 × 环境参数 × 当前目标品质警戒倍率`（单位/秒）。
- 环境参数可在配置面板中调节，默认值为 1.0。
- 环境参数 > 1 时警觉度增长更快，< 1 时更慢，可用于模拟不同难度环境。

### 2.8 偷窃计时

- 玩家 **长按 F** 开始偷窃时，计时器启动并开始累计时间（精确到小数点后1位，单位为秒）。
- 偷窃 **成功** 或 **失败** 时，计时器立即停止，保留最终时间。
- 切换偷窃目标（Q/E）时，计时器重置为 0 并停止计时，等待下次按 F 重新启动。
- 游戏重新开始时，计时器清零。
- UI 显示格式：未开始时显示“等待开始...”，计时中/结束时显示“XX.X秒”。

### 2.9 结果弹窗

- 偷窃 **成功** 时，弹出结果弹窗，显示偷窃用时，并提供“重新开始”按钮。
- 偷窃 **失败** 时（警觉度满 / 被NPC视野发现），弹出结果弹窗，显示偷窃用时，并提供“重新开始”按钮。
- 弹窗未关闭时，游戏逻辑暂停，玩家无法操作。
- 点击“重新开始”按钮后，重置所有状态并关闭弹窗。

## 三、偷窃失败条件（满足任一即失败）

| 序号 | 失败条件 |
|:---:|------|
| 1 | NPC警觉度达到上限值 |
| 2 | 玩家处于NPC锥形视野内且正在偷窃（长按F） |

---

## 四、按键操作一览

| 按键 | 操作方式 | 效果 |
|:---:|:---:|------|
| F | 长按 | 执行偷窃，增长偷窃进度；同时增长警觉度 |
| F | 松开 | 停止偷窃，进度缓慢回退，警觉度缓慢回退 |
| Q | 长按 | 切换至上一个道具目标，进度清零，增长警觉度 |
| E | 长按 | 切换至下一个道具目标，进度清零，增长警觉度 |

---

## 五、默认参数配置表

### 5.1 基础参数

| 参数名称 | 默认值 | 说明 |
|------|:---:|------|
| 偷窃速度 | 10 | 偷窃进度的基础增长速率（单位/秒） |
| 偷窃速度加成系数 | 1 | 始终生效的偷窃速度倍率乘数，不影响警觉度 |
| 环境参数 | 1.0 | 警觉度增长的额外环境系数，模拟不同难度环境 |
| 偷窃回退速度 | 3 | 松开F后偷窃进度的衰减速率（单位/秒） |
| 警戒速度 | 10 | 警觉度的基础增长速率（单位/秒） |
| 警戒值上限 | 100 | 警觉度满值，达到即偷窃失败 |
| 警戒回退速度 | 15 | 松开所有按键后警觉度的衰减速率（单位/秒） |

### 5.2 品质参数

| 品质 | 进度需求 | 警戒倍率 | 颜色标识 |
|:---:|:---:|:---:|:---:|
| 白色 | 40 | 1.0 | #ffffffff |
| 绿色 | 50 | 1.1 | #4CAF50 |
| 蓝色 | 65 | 1.25 | #2196F3 |
| 紫色 | 85 | 1.45 | #9C27B0 |
| 橙色 | 120 | 1.7 | #FF9800 |
| 未知 | 60 | 1.2 | #8d8d8dff |

> **计算公式**
> - 偷窃进度增长 = `偷窃速度 × 偷窃速度加成系数` /秒
> - 警觉度增长 = `警戒速度 × 环境参数 × 当前目标品质警戒倍率` /秒

## 六、边界条件与异常处理

| 场景 | 处理逻辑 |
|------|------|
| NPC身上无道具 | 不进入偷窃状态，提示玩家无可偷窃目标 |
| 偷窃进度回退至 0 | 进度值锁定为 0，不出现负值 |
| 警觉度回退至 0 | 警觉度锁定为 0，不出现负值 |
| 切换目标时仅剩1个道具 | Q/E切换仍指向该道具（循环不变） |
| 偷窃成功后 | 该道具从NPC身上移除，自动选中下一个道具；若NPC无剩余道具则判定全部偷窃完成 |
| 偷窃失败后 | 本局游戏结束，重置所有状态（进度归零、警觉度归零） |
| NPC视野边界判定 | 玩家恰好处于视野锥边缘时，按「在视野内」处理 |
| 同时按 Q/E 和 F | 以最后按下的按键为准，若最后为F则判定为偷窃中 |
| 偷窃速度加成系数为 0 | 偷窃进度不增长，但警觉度正常增长（惩罚性设计） |
| 环境参数为 0 | 警觉度不增长，可安全偷窃（调试用） |

## 七、游戏状态流转

```
[潜行中] --按F/Q/E--> [偷窃中]
   ↑                      |
   |---偷窃成功(进度达标)--|
   |---偷窃失败(警觉满/被看到)---|
   |                      |
   +--松手回退进度/警觉度--+
```

- **潜行中**：玩家未按任何操作键，可自由移动至NPC身后。警觉度和偷窃进度自然回退。
- **偷窃中**：玩家按住Q/E/F中任一按键，执行偷窃或切换操作。警觉度持续增长。
- **偷窃成功**：当前道具偷窃完成，道具转移至玩家背包。
- **偷窃失败**：触发任一失败条件，本局结束。
